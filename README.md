# IIT Kanpur Counselling Service Website

A responsive and dynamic website for the Institute Counselling Service (ICS), IIT Kanpur. It provides information, booking features, and forms to support student mental health and counseling outreach.

## 🚀 Features

- 🧠 **About ICS** – Learn about the Institute Counselling Service and its mission.
- 📋 **Appointment Booking** – Book one-on-one counseling sessions via a dynamic form.
- ✍️ **Feedback Form** – Share experiences anonymously or with contact details.
- 💡 **Therapy Services** – Overview of available services like group therapy, trauma care, and family counseling.
- 📍 **Contact Section** – Get in touch via email, phone, or visit the office directly.
- 🔄 **Fully Integrated Backend** – Form submissions are connected to MongoDB Atlas via Node.js + Express backend.
- 📄 **Deployed on Render** – Easily accessible via public Render URL.
